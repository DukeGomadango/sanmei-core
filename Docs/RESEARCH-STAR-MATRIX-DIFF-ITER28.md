## 4. `mock-v1` 行列との関係

現行 `SU_a_b` / `MA_a_b` は **星名を保持せず ID のみ**。公開表と突合するには、次のいずれかが必要になる。

1. **星名→ tenStarId の監修対応表**（`L0` 推奨）、または  
2. 公開表を **数値 ID に写像**した上で、既存行列と **セルごとに一致判定**するテストフィクスチャ。
